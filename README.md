## Installation

Get a free API Key [here](https://api.nasa.gov/).

Clone the repo
git clone https://github.com/toniwilliams1/NASA-Astronomy-Picture-Of-The-Day.git

Enter your API in script.js
const API_KEY = 'ENTER YOUR API';
